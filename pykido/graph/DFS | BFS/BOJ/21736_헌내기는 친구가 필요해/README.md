## 문제 해결의 흐름
### 1. 입력 받기 
#### (1) 캠퍼스의 크기 R, C 입력 받기 
#### (2) 캠퍼스 정보들 입력 받기. 이때, 도연이의 좌표 queue에 push
### 2. bfs 구현 
#### (1) 상하좌우 탐색하며 빈 공간 -> 이동, 사람 -> 이동, 벽 -> 이동 X
#### (2) 사람 만난 수를 전부 cnt에 저장
#### (3) 이동하지 못한 벽이나 이동하였던 곳은 전부 방문 처리
#### (4) 더 이상 이동할 수 없다면 bfs 로직이 끝날 것이다.
