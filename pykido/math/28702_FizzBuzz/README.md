# [Bronze 1] FizzBuzz - 28702

[문제링크](https://www.acmicpc.net/problem/28702)
## 문제 해결의 흐름
### 1. 3개의 문자열을 차례대로 받아야한다.
### 2. 문자열이 숫자인지 찐 문자열인지 판단이 필요함.
### 3. 입력 받은 문자열을 숫자로 변환했을 때 무슨 수인지 확인
### 4. 3개의 문자열값들을 보고 다음에 올 숫자 예측
- only 3의 배수 : "Fizz" 출력
- only 5의 배수 : "Buzz" 출력 
- 3의 배수 & 5의 배수 : "FizzBuzz" 출력
