# [Bronze 4] welcom kit - 31403

[문제링크](https://www.acmicpc.net/problem/30802)

## 문제 해결의 흐름
### 1. 입력 받기 
- 첫번째 줄 : 참가자 수 
- 두번째 줄 : 티셔츠 사이즈별 신청자수
- 세번째 줄 : 티셔츠 묶음 수 T, 볼펜 묶음 수 P
### 2. 주문하기 - 첫번째 줄 (티셔츠를 T장씩 몇 묶음 주문할까? 티셔츠 남아도 됨) 
- logic) 0 ~ T : 1장 주문, T + 1 ~ 2 * T : 2장 주문....
- % == 0 : 몫 만큼 주문 
- % != 0 : 몫 + 1 만큼 주문
### 3. 주문하기 - 두번째 줄 (펜은 몇 자루 주문할까? 남거나 부족해선 안됨)
- 참가자 수 / P , 참가자 수 % P 출력
